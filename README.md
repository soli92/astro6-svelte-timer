# ⏱️ Timer Cosmico

Un timer simpatico e animato costruito con **Astro** e **Svelte**.

## ✨ Features

- 🌀 Anello di progresso SVG animato con colori dinamici
- 😄 Emoji mood che cambia in base al tempo rimasto
- 🎉 Pioggia di coriandoli al termine del timer
- ⚠️ Avviso urgente negli ultimi 10 secondi
- 🎨 UI dark glassmorphism con sfondo cosmico
- 📱 Completamente responsive

## 🚀 Avvio locale

```bash
npm install
npm run dev
```

## 🛠️ Tech Stack

- [Astro](https://astro.build/) — Framework web
- [Svelte 5](https://svelte.dev/) — Componente timer reattivo
- CSS puro — Animazioni e glassmorphism
